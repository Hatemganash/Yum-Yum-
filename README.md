# Yum-Yum-
Hello

Yum Yum! Food App provides users with different categories of food, details of each dish and add it to the order list.

Technologies:
UlKit
MVC Architecture Pattern
Networking using URLSession
UserDefaults
Kingfisher
IQKeyboardmanager
KRProgressHUD
Storyboard, Nibs
